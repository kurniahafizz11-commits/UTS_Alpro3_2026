## 📋 Deskripsi

Repositori ini merupakan repositori resmi pengumpulan tugas **Ujian Tengah Semester (UTS) Algoritma dan Pemrograman** untuk lab 3, Semester Genap T.A. 2025/2026. Setiap praktikan **wajib melakukan fork** terhadap repositori ini dan mengumpulkan jawaban melalui fork tersebut.

---

## 📂 Struktur Repositori

Setelah melakukan fork, buat struktur repositori seperti ini:

```
NIM_NamaLengkap/
├── soal1.cpp       
├── soal2.cpp
├── soal3.cpp
└── soal4.cpp
```

> ⚠️ Pastikan nama folder utama menggunakan format **`NIM_NamaLengkap`** (contoh: `231401032_ParulianDwiReslia`)

---

## 📦 Instruksi Pengumpulan

### 1. Fork Repositori
Klik tombol **Fork** di pojok kanan atas halaman ini untuk membuat salinan repositori ke akun GitHub kamu.

### 2. Clone ke Lokal
```bash
git clone https://github.com/<username-kamu>/<nama-repo>.git
cd <nama-repo>
```

### 3. Isi dengan kode solusi
Isi repositori dengan kode solusi untuk keempat soal.

### 4. Commit & Push
```bash
git add .
git commit -m "NIM_NamaLengkap - Submit UTS AP3"
git push origin main
```

### 5. Struktur & Dokumentasi Kode
- Kode ditulis **rapi dan konsisten**.
- Di **baris paling bawah** setiap file, sertakan **algoritma/logika penyelesaian dalam bentuk komentar** yang menjelaskan langkah-langkah secara sistematis.

### 6. Flowchart (Wajib)
Buat **4 flowchart** (satu per soal) menggunakan draw.io, Lucidchart, Canva, atau tools lainnya. Simpan dalam format **PNG/JPG atau PDF**.

### 7. Video Demonstrasi (Wajib)
Buat **video** (maksimal **20 menit**) yang:
- Menjelaskan cara kerja kode untuk setiap soal.
- Mendemonstrasikan **semua test case** dengan output yang benar.

### 8. Upload Video & Flowchart
- Buat folder di **Google Drive** yang sudah disediakan dengan nama: `NIM_NamaLengkap_UTS_AP3`
- Gabungkan video dan 4 flowchart ke dalam folder tersebut.
- Pastikan akses folder diatur ke **"Anyone with the link can view"**.

### 9. Pengumpulan Akhir
Kirimkan **tautan repository GitHub hasil fork** kamu melalui spreadsheet yang akan diberikan oleh asisten.
